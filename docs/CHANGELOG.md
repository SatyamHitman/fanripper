# Changelog

## 0.1.30
- **Embedded metadata** — creator, caption, PPV price and original post date are
  written into each downloaded file's own metadata.
- **Richer delivery captions** — the post / chat text now travels with the file.
- **Faster, more reliable bulk** — parallel download engine, auto-resume after a
  crash, and an instant Stop.
- **Large-file + cloud reliability** — fixes for multi-gigabyte DRM videos and
  cloud backup.

Older builds are distributed via https://install.fanripper.com. The extension
auto-updates once installed.
